position

.positionRelative {position:relative;} - 해당영역을 차지
.positionAbsolute {position:absolute;} - 해당영역을 차지하지 않고 레이어 위에 올릴때

border 디자인

.boxRadius {border-radius:5px;} - 레이어 라운드 5px
.boxShadow {box-shadow:1px 1px rgba(0, 0, 0, 0.1); -webkit-box-shadow:1px 1px rgba(0, 0, 0, 0.1); -moz-box-shadow:1px 1px rgba(0, 0, 0, 0.1);} - 레이어 그림자 효과

기타

.hidden {display:none;} - 숨기기
.b {font-family:nanumBold;} - 글자 굵게
.fl {float:left;} - 왼쪽정렬
.fr {float:right;} - 오른쪽 정렬
.cb {clear:both;} - 아래로 내리기
.left {text-align:left;} - 글자 왼쪽 정렬
.center {text-align:center;} - 글자 센터 정렬
.right {text-align:right;} - 글자 오른쪽 정렬
.fcenter {margin: 0 auto;} - 레이어 가운데 정렬
